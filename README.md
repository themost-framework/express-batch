# @themost/express-batch

[![npm version](https://badge.fury.io/js/%40themost%2Fexpress-batch.svg)](https://www.npmjs.com/package/@themost/express-batch)

This module provides a batch processing middleware for `@themost/express` services. It allows you to define batch operations that can be executed in a single request, improving performance and reducing the number of round-trips to the server.
